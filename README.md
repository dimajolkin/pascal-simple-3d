# Работа над 3d графиков в паскале, с университета 2013/2014

Для запуска: https://pascalabc.net/ssyilki-dlya-skachivaniya

Для запуска Shift + F9

1. вкл/выкл - проекций
2. вкл/выкл - перспективы
3. w,a,s,d - выбор оси вращения, space - вращение
